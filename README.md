# 贪吃蛇小游戏

一个纯原生 HTML/CSS/JavaScript 的贪吃蛇小游戏，开箱即玩。

## 玩法

- 点击 `开始游戏` 开始挑战。
- 可在开始前选择 `难度` 与 `主题`，难度越高移动越快且同一道具得分更低。
- 键盘支持：方向键、`WASD` 控制方向，空格或 `P` 暂停/继续，`R` 重新开始，`H` 打开/收起玩法说明。
- 触屏支持：页面下方方向按钮可直接控制。
- 道具类型：
  - `🍎` 苹果：基础得分 10
  - `🍌` 香蕉：基础得分 15
  - `💣` 炸弹：扣分并减少 1 点生命，生命耗尽即结束
- 炸弹出现时会与可得分道具同时出现，吃掉任意一个后会刷新下一波道具。
- 撞墙或咬到自己即游戏结束，可点击 `重新开始` 立刻再来一局。
- 页内提供“游戏面板 / 玩法说明”选项卡，随时可回看规则与快捷键。

## 本地运行

直接双击打开 `index.html`，或在当前目录启动任意静态服务器后访问页面。
